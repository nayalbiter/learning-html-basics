The position property specifies the type of positioning method used for an element (static, relative, absolute, fixed, or sticky).

Article 1

This is the default value for elements. The element is positioned according to the normal flow of the document. The left, right, top, bottom and z-index properties do not affect an element with position: static. Let's use an example to show that position: static has no effect on the position of an element. We have three divs placed in a parent container. We'll use this example throughout this article.
